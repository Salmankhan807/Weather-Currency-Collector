#  Weather & Currency Collector

## About the Project

I built this project to strengthen my Python programming skills and gain hands-on experience with Object-Oriented Programming (OOP), working with REST APIs, JSON parsing, and modular programming.

The application allows users to check the live weather of any city and convert currencies using real-time exchange rates. This portfolio project  marks an important milestone in my journey toward becoming a Data Analyst and Data Scientist."

---

# Features

# Weather Module

- Search weather by city
- Displays:
  - Temperature
  - Feels Like Temperature
  - Humidity
  - Wind Speed
  - Weather Condition
- Handles invalid city names gracefully

## Currency Converter

- Convert between any two supported currencies
- Uses real-time exchange rates
- Handles invalid currency codes
- Validates user input

## Installation

Clone the repository

```bash
git clone https://github.com/Salmankhan807/Weather-Currency-Collector.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the project

```bash
python main.py
```
